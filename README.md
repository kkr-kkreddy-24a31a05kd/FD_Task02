# FD_Task02
The updated calculator design brings a much more modern and polished feel to your project. Here is a description of the creative improvements I implemented:

Dark Mode & Glassmorphism: I shifted the aesthetic to a sophisticated dark theme (slate-950). By using a "glass" panel effect with a backdrop blur and semi-transparent backgrounds, the calculator feels like a modern application window rather than a basic web form.

Tactile Feedback (Micro-interactions): I added a btn-press transition effect. When you click a button, it subtly shrinks (scaling to 0.95), giving the user satisfying physical feedback that makes the interface feel responsive and alive.

Refined Color Palette: Instead of standard browser buttons, I used a professional palette from Tailwind CSS. The operators (like +, -, ×, /) are highlighted in bold indigo, the equals button is a distinct emerald green, and the clear button is a red accent, making the calculator intuitive and easy to navigate.

Clean, Modern Typography: By using lighter font weights and generous spacing, the layout feels less cluttered and more "premium," matching current design trends for web and mobile apps.
